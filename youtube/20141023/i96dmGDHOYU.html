Uploaded on Jun 21, 2010

Djokovic v Rochus, Wimbledon 2010, 4th set, 10th game, Djokovic serving for the set and struggling.

In this extremely boring video, you can see that my version of BetAngel Tennis Trader for The Toy updates the scores (taken from the SlamTracker scoreboard) and odds (taken from The Toy) automatically (it's not the case in the original version of Tennis Trader). Note that having the SlamTracker window open is not neccessary.

You can see the Tennis Trader predictions for the next point and after each point you can check if it was accurate. At the end of the video, you can also see that Tennis Trader can determine the starting price of the favorite player.

Truth be told, this was a boring game. The next two games in the 5th set were much more interesting as both players broke each others serve (15-40) and the odds moved from 1.4 to 2.4 and seeing Tennis Trader predictions for points in these two games was much more useful than in the game shown in this video.
